Общее

Данная СЭД предназначена для автоматизации электронного докумнетооборота между университетами, путем заверения документов электронной подписью.

Предусмотрена функция предварительного просмотра, используя встроенный просмотр pdf-файла для дальнейшего его скачивания.

Документы хранятся исключительно в зашифрованном виде.

Входящие

На данной странице отображаются все электронные документы, отправленные пользователю для проверки.

Также присутствует таблица c такими полями как: "Название документа", "Email отправителя", "Скачать", "Подписать", "Отказать".

Пользователю необходимо проверить документ и при условии полного соответствия нажать кнопку "Подписать", либо "Отказать". После чего документ отправляется пользователю, который подал его на подписание.

Отправленные

На данной странице отображаются все электронные документы, отправленные данным пользователем для дальнейшего рассмотрения и получения результата "Подписано" либо "Отказано".

Также присутствует таблица c такими полями как: "Название документа", "Email отправителя", "Скачать".

Подписанные

На данной странице отображаются все электронные документы данного пользователя, которым был присвоен статус "Подписано" при помощи электронной подписи.

Также присутствует таблица c такими полями как: "Название документа", "Email отправителя", "Скачать".

После того, как пользователь, которому отправлялся документ на проверку, подписал документ, на последнюю страницу добавляется уникальная электронная подпись, принадлежащяя исключительно данному пользователю и сведетельствующая о том, что документ принят.

Отказанные

На данной странице отображаются все электронные документы данного пользователя, которым был присвоен статус "Отказано" при помощи электронной подписи.

Также присутствует таблица c такими полями как: "Название документа", "Email отправителя", "Скачать".

После того, как пользователь, которому отправлялся документ на проверку, подписал документ, на последнюю страницу добавляется уникальная электронная подпись, принадлежащяя исключительно данному пользователю и сведетельствующая о том, что документ отказан.

Отправить документ

На данной странице реализована функция отправки документа определенному пользователю.

Также присутствует элемент, позволяющий выбрать документ с локального хранилища и выбор отправителя с кнопкой отправить.

Контрагенты

На данной странице отображается информация о контрагентах.

Также присутствует таблица c такими полями как: "Email", "Университет", "Факультет", "ФИО", "ИНН", "Адрес".
